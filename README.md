# Important!
For this plugin/mod to work, you'll need to ensure your BepInEx is configured to have *`HideManagerGameObject`* set to *`true`*, or else the game will destroy the plugin!

# About
Blood Fueled Enemies is a simple mod which adds a cheat which does (kinda) what its name suggests, when *you* get hurt, enemies will use your blood (currently not visualized) to heal. The mod also features a couple of configuration options to control how this works.
